# aws-cert-portfolio

AWS certification prep + portfolio artefacts focused on cloud security.

## Structure
- **ccp-clf-c02/** — Cloud Practitioner (foundational): exam blueprint, notes, small labs, artefacts
- **saa-c03/** — Solutions Architect Associate: secure architecture labs + IaC artefacts
- **security-specialty-scs-c02/** — Security Specialty: logging/detection/IR labs + runbooks
- **diagrams/** — diagrams used across artefacts
- **templates/** — reusable templates (runbooks, one-pagers, review logs)

## Hygiene
No secrets are committed (keys/tokens/credentials). Terraform state is ignored.
