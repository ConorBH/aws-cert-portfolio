# aws-cert-portfolio

Portfolio + study notes for AWS certifications, with hands-on artefacts that prove applied cloud security skills.

## What this repo contains
- **ccp-clf-c02/** — Cloud Practitioner (foundational): exam blueprint, notes, small labs, and artefacts
- **saa-c03/** — Solutions Architect Associate: secure architecture labs + IaC artefacts
- **security-specialty-scs-c02/** — Security Specialty: logging/detection/IR labs + runbooks
- **diagrams/** — architecture + investigation flow diagrams
- **templates/** — reusable templates (runbooks, one-pagers, review logs)

## Safety / hygiene
- No secrets are ever committed (keys, tokens, credentials). See `.gitignore`.
